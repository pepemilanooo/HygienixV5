# Hygienix - Gestionale SaaS Disinfestazioni

## Quickstart Railway
1. Unzip → GitHub repo "hygienix"
2. Railway New Project → GitHub
3. Provision Postgres → Copy DATABASE_URL to vars
4. npm install → npx prisma db push → Deploy!
